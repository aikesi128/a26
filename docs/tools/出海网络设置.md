>  运营Facebook需要一个纯净的海外住宅ip, 所以单单翻墙出去是不行的. 还需要纯净的住宅ip, 避免账号被封.



- 购买的新加坡服务器节点, 网络稳定(月租28rmb), 搭建openvpn服务, 本地通过openvpn客户端进行连接, 
- 在服务器上搭建vpn的时候, dns选择本地的dns就行.
- 客户端连接后使用谷歌浏览器访问外网, safari不知道为什么可能会很慢.
- 购买海外代理ip, 结合adspower这个软件, 就可以伪造出一个纯净的海外住宅ip. 因为代理ip在国内不提供直连服务, 所以必须保证本地的网络环境是可以访问外网的.
