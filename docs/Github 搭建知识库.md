



## Github 搭建知识库

1. 在github新建一个仓库, 设置为public类型.

2. 设置当前仓库让github托管. (分支设置为main, 路径为/root)

3. 在仓库本目录新建_config.yml文件

   ```
   theme: jekyll-theme-minimal # For supported themes
   remote_theme: just-the-docs/just-the-docs # For other Jekyll themes hosted on GitHu
   url: "https://aikesi128.github.io" # Your site's base URL
   baseurl: "/a26" # 这里要设置为当前仓库的名称, 否则路径会查找错误
   ```
